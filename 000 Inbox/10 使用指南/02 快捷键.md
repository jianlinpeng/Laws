### 第一部分：工作区快捷键

|        | 功能          | 快捷键             |
| ------ | ----------- | --------------- |
| 工作区快捷键 |             |                 |
|        | 打开设置        | cmd ，           |
|        | 向左折叠        | option shift ←  |
|        | 向右折叠        | option shift →  |
|        | 放大          | Cmd =           |
|        | 缩小          | Cmd -           |
|        | 原始大小        | Cmd + 0         |
|        | 关系图谱        | Cmd + G         |
| 文件快捷键  |             |                 |
|        | 打开命令面板      | Cmd O           |
|        | 打开快速切换      | Cmd P           |
| 编辑快捷键  |             |                 |
|        | 切换编辑模式/阅读模式 | Cmd E           |
|        | 切换实时编辑/源码模式 | Cmd Shift E     |
|        | 增加笔记属性      | Cmd ;           |
|        | 设置标题        | Cmd 1/2/3/4/5/6 |
|        | 删除标题格式      | Cmd Backspace   |
|        | 加粗/取消加粗     | Cmd B           |
|        | 高亮/取消高亮     | Cmd H           |
|        | 斜体/取消斜体     | Cmd I           |
|        | 插入引用        | Cmd Option Q    |
|        | 插入链接        | Cmd L           |
|        | 插入附件        | Cmd Option A    |
|        | 插入题注        | Cmd Option B    |
