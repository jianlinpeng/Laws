## **一、第一届至第五届（1954-1975 年）​**​

| ​**任期**​              | ​**总理**​ | ​**核心议题**​                             |
| --------------------- | -------- | -------------------------------------- |
| ​**第一届（1954-1959）​**​ | 周恩来      | 确立政务会议制度，制定新中国首批行政规章，推动计划经济体系构建        |
| ​**第二届（1959-1965）​**​ | 周恩来      | 应对自然灾害与苏联撤资，调整国民经济计划（如“调整、巩固、充实、提高”方针） |
| ​**第三届（1965-1975）​**​ | 周恩来      | 文革期间维持经济基本运行，推动三线建设，保障国防与重工业发展         |

​**特点**​：以政务会议为核心，集体决策为主，议题集中于计划经济与国家安全。

---

## ​**二、第六届至第十届（1975-2003 年）​**​

|​**任期**​|​**总理**​|​**核心议题**​|
|---|---|---|
|​**第六届（1975-1978）​**​|周恩来（至 1976 年）  <br>华国锋（1976 年后）|恢复国民经济，结束文革混乱局面|
|​**第七届（1978-1983）​**​|华国锋、赵紫阳|改革开放初期政策设计（如农村家庭联产承包责任制、经济特区设立）|
|​**第八届（1983-1988）​**​|赵紫阳|深化经济改革，推动价格双轨制与国企改革试点|
|​**第九届（1993-1998）​**​|李鹏|应对通货膨胀，实施“软着陆”政策，建立现代企业制度|
|​**第十届（1998-2003）​**​|朱镕基|国企改革攻坚（如“抓大放小”），加入 WTO 前的法规体系调整|

​**特点**​：经济体制改革为核心，国务院全体会议地位逐渐弱化，常务会议决策权增强。

---

## ​**三、第十一届至第十四届（2003 年至今）​**​

| ​**任期**​               | ​**总理**​ | ​**核心议题**​                           |
| ---------------------- | -------- | ------------------------------------ |
| ​**第十一届（2003-2008）​**​ | 温家宝      | 应对 SARS 疫情，取消农业税，推动社会保障体系初步建设          |
| ​**第十二届（2013-2018）​**​ | 李克强      | 简政放权（取消下放行政审批 215 项），“双创”政策（大众创业、万众创新） |
| ​**第十三届（2018-2023）​**​ | 李克强      | 深化“放管服”改革，应对中美贸易摩擦，推动减税降费与稳就业政策      |
| ​**第十四届（2023-至今）​**​   | 李强       | 推进城市更新与乡村振兴，加强生态环境保护，推动数字经济与绿色能源发展   |
